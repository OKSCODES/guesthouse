# Umpohliew Guest House

Production-ready React + Vite + Tailwind website with Firebase booking/contact/review backend.

## Run
```bash
npm install
cp .env.example .env
npm run dev
```

## Firebase collections
- bookings
- contacts
- reviews

Enable Firestore in Firebase Console and paste config values into `.env`.

## Admin
Set `VITE_ADMIN_EMAIL` and `VITE_ADMIN_PASSWORD` in `.env`, then visit `/admin/login`.
